# Recipes

Recipes are real Hollywood use cases we care about in the Dedalus monorepo.

| Recipe                                  | What it proves                                                                  |
| --------------------------------------- | ------------------------------------------------------------------------------- |
| [S3 Cache](s3-cache.md)                 | A script can hit S3-compatible object storage locally, then run through GitHub. |
| [Package Artifact](package-artifact.md) | A privileged infrastructure action can be typed without shell in YAML.          |

The maintained source files live under `examples/`. Recipe pages explain the
why and show the generated GitHub Actions surface.
